## Prerequisites

Before running the Deliveries API, make sure you have the following:

- Java Development Kit (JDK) 17 or later installed on your machine.
- Gradle build tool for Java.
- An integrated development environment (IDE) such as IntelliJ or Eclipse.

## Installation

1. Clone this repository to your local machine.
2. Import the project into your IDE as a Gradle project.
3. Build the project using Gradle to resolve dependencies.
4. Configure the necessary database connection in the application properties file.
5. Run the application using your IDE or Gradle command line.
